Resistor Color Code

Read a resistor's color bands as ohms and tolerance, or turn a value back into bands. Supports 4- and 5-band resistors. Runs locally in your browser.

Pick bands to read the value.

Encode a value to bands

Digits 0–9 are black, brown, red, orange, yellow, green, blue, violet, gray, white. The multiplier band is the same sequence (×10ⁿ) plus gold (×0.1) and silver (×0.01) for values below 1 Ω. The tolerance band gives the manufacturing precision: gold ±5% and silver ±10% are most common; brown ±1% and red ±2% for precision parts. Value = (digits as a number) × 10^multiplier. Runs locally — nothing leaves your browser.
