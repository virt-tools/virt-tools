CSV Sort & Multi-Column Sort

Sort CSV data by one or more columns with configurable sort order, numeric or string comparison, and case sensitivity. Add multiple sort levels (primary, secondary, etc.) for multi-key sorting. Auto-sniffs the delimiter, handles quoted fields and embedded newlines. Pairs with the CSV Viewer, CSV Filter, and CSV Deduplicator. Everything runs locally in your browser.
