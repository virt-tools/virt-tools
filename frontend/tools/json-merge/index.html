JSON Merger

Deep-merge two or more JSON documents into one. Objects are merged key-by-key (later documents win on conflicts); arrays are concatenated, replaced, or merged by an index or key. Choose indentation and whether to sort keys. Pairs with the JSON Formatter, JSON Flatten, JSON Sort, and JSON Diff tools. Everything runs locally in your browser; nothing is uploaded.

Deep merge: nested objects are combined recursively; primitives and arrays follow the chosen strategy. Null values in later documents delete the key unless "keep null" is checked. Pairs with the JSON Formatter, JSON Flatten, JSON Sort, and JSON Diff tools. Runs locally.
