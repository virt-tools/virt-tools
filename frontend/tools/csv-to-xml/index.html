CSV to XML Converter

Convert CSV data into XML. Each row becomes a child element under a root, with column names as element tags. Handles quoted fields, embedded commas and newlines, ragged rows, and XML-special character escaping. Choose the delimiter, root element name, row element name, and whether the first row is a header. Pairs with the XML Formatter and CSV to JSON. Everything runs locally in your browser.

Column names are sanitized to valid XML tag names (spaces and special chars → underscores, leading digits prefixed). Characters &, <, >, " are escaped. Use "attributes" mode to put column values in attributes instead of child elements. Pairs with the XML Formatter and CSV to JSON. Runs locally.
