WPM uses the standard 5-characters-per-word definition: gross WPM = (characters typed ÷ 5) ÷ minutes; net WPM subtracts errors. Accuracy = correct characters ÷ total typed. The timer starts on the first keystroke and stops when your input reaches the end of the text. Runs locally.
