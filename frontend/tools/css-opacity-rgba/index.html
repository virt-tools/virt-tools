CSS Opacity & RGBA Calculator

Convert a hex color + opacity value to CSS rgba(), hsla(), and 8-digit hex (#RRGGBBAA) formats. Adjust the alpha slider, preview the color over different backgrounds, and copy the ready-to-paste CSS. Pairs with the Color Converter and Contrast Checker. Everything runs locally in your browser.

White background:
rgba(110, 168, 254, 0.5)
Black background:
rgba(110, 168, 254, 0.5)
Checkerboard (transparency):
rgba(110, 168, 254, 0.5)

The alpha slider controls the opacity from 0 (fully transparent) to 1 (fully opaque). The 8-digit hex form (#RRGGBBAA) is supported in all modern browsers. Pairs with the Color Converter and Contrast Checker. Runs locally.
