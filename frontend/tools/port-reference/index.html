Network Port Reference

Common TCP and UDP port numbers and the services that run on them — from the well-known range (SSH, HTTP, DNS, SMTP) through the registered ports used by databases, caches, and apps. Filter by port, name, or protocol; click a row to copy. Runs locally in your browser.

PortProtoServiceDescription

Covers well-known ports (0–1023, assigned by IANA) and common registered ports: web, mail, directory, databases, remote access, caching, streaming, and more. Protocol shows TCP, UDP, or both (a few services use both transports, like DNS and NFS). Port numbers are facts; for an authoritative list see the IANA Service Name and Transport Protocol Port Number Registry. Click any row to copy "port/proto name". Runs locally — nothing leaves your browser.
