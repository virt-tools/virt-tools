Playfair Cipher

Encrypt and decrypt with the Playfair cipher, the famous digraph substitution cipher invented in 1854 and used militarily through World War I and World War II (it was the first practical digraph cipher). It uses a 5×5 key square built from a keyword (with J merged into I), and encrypts pairs of letters at a time rather than single letters — so it defeats simple frequency analysis. Three rules handle each pair: same row → shift right, same column → shift down, otherwise rectangle → swap columns. Identical letters in a pair are split with an X, and an odd-length message is padded with X. Enter a keyword and text to encipher or decipher, with the key square and prepared digraphs shown for reference. Everything runs locally in your browser.

5×5 key square

The math. The key square is a 5×5 grid filled from the keyword (upper-cased, J→I, duplicates removed) followed by the remaining letters of ABCDEFGHIKLMNOPQRSTUVWXYZ in order — 25 letters, since J shares I's cell. Plaintext is prepared into digraphs: upper-case, J→I, non-letters dropped, then split into pairs; if a pair's two letters are identical an X is inserted between them, and a trailing lone letter is padded with X. Each pair (a, b) at positions (r1,c1) and (r2,c2) is encrypted as: if r1 == r2 (same row) take the letter to the right of each, wrapping; if c1 == c2 (same column) take the letter below each, wrapping; otherwise (rectangle) replace a with the letter in a's row and b's column, and b with the letter in b's row and a's column. Decryption reverses the row/column shifts (left/up instead of right/down); the rectangle rule is its own inverse. Because the cipher operates on digraphs, common-letter frequencies are flattened, which is why it resisted manual cryptanalysis for decades. Pairs with the Caesar, Vigenère, Beaufort, Bacon, Columnar Transposition, and Classical Ciphers tools. Everything runs locally — nothing leaves your browser.
