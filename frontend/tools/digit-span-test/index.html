Digit Span Memory Test

Test your verbal working memory. A sequence of digits flashes on screen for a few seconds; when it hides, type back exactly what you saw. Each level adds one digit. Two trials per level — pass at least one to advance, fail both and the test ends with your best span. Adjust the display time and starting length. Everything runs locally in your browser.

How it works: the digits show for (display time × length), then hide and the input opens. Type the sequence in order — spacing and dashes are ignored, so "1 2 3" matches "123". Each level is one digit longer than the last. Two trials per level: pass one to advance, fail both and the test ends, reporting your best span (longest sequence you reproduced correctly). Average adult digit span is about 7 ± 2. Runs locally — no data leaves your browser.
