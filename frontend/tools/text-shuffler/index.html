Text Shuffler & Scrambler

Shuffle or scramble text six ways — shuffle words within each line, shuffle characters across the whole text, shuffle characters within each word, scramble each word into a typoglycemic anagram (keep the first and last letter, jumble the middle — the trick that supposedly keeps text readable), shuffle lines, and apply random letter case. Optional numeric seed for reproducible output. Pairs with the Text Reverser and Case Converter. Runs locally in your browser.

Each button applies to the current input. The optional seed makes a shuffle reproducible — the same seed and input always produce the same output. Runs locally.
