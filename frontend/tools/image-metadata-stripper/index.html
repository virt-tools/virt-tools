Image Metadata Stripper

Remove EXIF, GPS, XMP, IPTC and other embedded metadata from an image by re-encoding it — so location data, camera info, and timestamps don't leak when you share a photo. The image is decoded and re-drawn to a canvas entirely in your browser; nothing is uploaded.

The image is decoded and re-drawn to a fresh canvas, then re-encoded — canvas output never carries the original EXIF/GPS/XMP/IPTC/ICC, so the downloaded copy is metadata-free. Re-encoding to JPEG/WebP is lossy; PNG is lossless (and larger). Animated GIF/WebP collapse to a single frame; color profiles are not preserved. No network requests are made.
