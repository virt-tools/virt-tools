SRI Hash Generator

Generate Subresource Integrity (SRI) hashes for <script> and <link> tags so browsers refuse to run a script or stylesheet whose hash doesn't match. Drop one or more files (or paste the URL of a cross-origin asset you've already mirrored locally) and copy the ready-to-paste tag with its integrity attribute. Hashes are computed with the Web Crypto API; your files never leave the browser.
