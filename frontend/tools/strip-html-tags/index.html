Strip HTML Tags

Paste HTML and get the plain text back with all tags removed. Optionally convert block elements and <br> to line breaks, decode HTML entities, collapse runs of whitespace, and extract the list of links. Useful for cleaning up copied web content, email HTML, or rich-text exports. Everything runs locally in your browser.

Paste HTML to see output.

Tags are removed with a regex pass; <script>, <style>, and HTML comments are dropped entirely before stripping. With Line breaks on, closing </p>, </div>, </li>, </h1>… and <br> become newlines so paragraphs stay on separate lines. Decode entities turns &amp;→&, &copy;→©, numeric &#65;→A, etc. Collapse whitespace merges runs of spaces and blank lines. Any <a href> links found are listed below with their text and URL. Runs locally — nothing leaves your browser.
