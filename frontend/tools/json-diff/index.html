JSON Diff

Compare two JSON documents structurally and see exactly what changed, added, or removed — at each JSON path. Unlike a line diff, this understands the structure: re-ordered object keys count as unchanged, array-element changes are attributed to the right index, and nested paths like users[1].name are reported precisely. Everything runs in your browser; nothing is uploaded.

Objects are always compared key-order-insensitively (JSON object order is not semantic). With Ignore array element order on, arrays are compared as multisets of their canonicalized elements. Identical inputs report no changes. No network requests are made.
