CSV Merger

Merge (concatenate) multiple CSV files or text blocks into one. Add as many CSV inputs as you need, choose whether to keep the header from the first file only (avoiding repeated headers), and optionally align columns by header name. Handles quoted fields and embedded newlines. Pairs with the CSV Viewer, CSV Sort, and CSV Splitter. Everything runs locally in your browser.

"Keep first header only" strips the header row from subsequent CSVs. "Align by header name" reorders columns so matching headers line up; missing columns in a CSV are left empty. Pairs with CSV Viewer, CSV Sort, and CSV Splitter. Runs locally.
