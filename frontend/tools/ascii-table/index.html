ASCII Table Reference

Every one of the 128 ASCII characters with its decimal, hex, octal, 7-bit binary code, the character itself, its HTML entity, and — for control characters — the standard name and description. Filter by code, name, or character; hide control or printable ranges. Click a row to copy. Runs locally in your browser.

Standard 7-bit ASCII (ANSI X3.4 / ISO 646) covers codes 0–127. Codes 0–31 and 127 are control characters; 32 is space; 33–126 are printable. Named HTML entities (&lt;, &amp;, &quot;, &apos;, &gt;) are used where defined, otherwise the numeric form (&#N;) is shown. Click any value in a row to copy it.
