A decibel is a logarithmic ratio. Power ratios use dB = 10·log₁₀(P₂/P₁) — so doubling power is +3.01 dB, tenfold is +10 dB. Voltage / amplitude ratios (with the same impedance) use dB = 20·log₁₀(V₂/V₁) — doubling voltage is +6.02 dB. dBm is power relative to 1 mW: 0 dBm = 1 mW, 30 dBm = 1 W; dBW is relative to 1 W. Independent power contributions add directly in dB. Runs locally — nothing leaves your browser.
