HTML to Markdown Converter

Convert HTML into clean Markdown — headings, paragraphs, bold/italic/strikethrough, inline code, fenced code blocks, links, images, ordered and unordered lists (including nested), blockquotes, and horizontal rules. The reverse of the Markdown previewer. Everything runs in your browser; nothing is uploaded.

A small pure-JS parser walks the HTML structure (it does not depend on the DOM). Inline whitespace is collapsed; code blocks and <pre> content are preserved verbatim. Unknown tags are passed through as their text content. No network requests are made.
