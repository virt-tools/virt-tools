Text Repeater

Repeat a block of text any number of times with a custom separator — line, comma, space, tab, custom string, or none. Number each repetition, trim whitespace, and reverse the output. Useful for generating test data, building padded strings, creating repeated patterns, and bulk text generation. Pairs with the Lorem Ipsum and Mock Data generators. Everything runs locally in your browser.

The separator goes between repetitions. "Repeat each line independently" repeats every line of input separately, preserving their order. Runs locally.
