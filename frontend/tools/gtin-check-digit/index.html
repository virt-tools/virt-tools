EAN / UPC / GTIN Check Digit

Calculate or validate the check digit of an EAN-8, EAN-13, UPC-A, or GTIN-14 product barcode using the standard mod-10 alternating-weight algorithm (weights 3 and 1, rightmost data digit weighted 3). Companion to the EAN/UPC Barcode Generator. Runs entirely in your browser.

Spaces and dashes are ignored. Enter a full code (with check digit) to validate, or one digit short to just calculate the check digit.
