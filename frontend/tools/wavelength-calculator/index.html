For an electromagnetic wave in a vacuum (or air, close enough), wavelength λ = c / f with c = 299,792,458 m/s. The quarter-wave (λ/4) and half-wave (λ/2) lengths are the standard monopole and dipole antenna sizes. The ITU band names follow the standard 3:1 ratio table — VHF is 30–300 MHz (FM radio, TV), UHF 300–3000 MHz (TV, mobile, 2.4 GHz WiFi), SHF 3–30 GHz (5 GHz WiFi, satellite, radar), EHF 30–300 GHz (millimetre wave).
