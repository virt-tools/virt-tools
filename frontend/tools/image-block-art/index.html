Image to Block Art

Turn an image into Unicode block art using the full block character set (█▉▊▋▌▍▎▏ and the lower-half / upper-half / left / right blocks). Each row of pixels maps to one character — the character encodes the brightness from full to empty, giving higher vertical resolution than ASCII art. Pick a ramp, set the output width, and invert for light backgrounds. Copy or download the result. The image is processed on a local canvas — nothing is uploaded. Pairs with the ASCII Art and Braille Art generators.
