Transform origin

Functions are emitted in the order perspective → translate → rotate → rotateX → rotateY → scale → skewX → skewY and only when they differ from the identity, so a box you haven't touched produces transform: none;. transform-origin: center is omitted by default. Perspective applies a vanishing point for the 3D rotations. No network requests are made.
