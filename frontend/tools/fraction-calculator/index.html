Fraction Calculator

Add, subtract, multiply, and divide fractions exactly — no floating-point rounding. Enter each value as a fraction (3/4), a mixed number (1 1/2), a decimal (0.125), or a whole number (5), and the result is shown as a simplified improper fraction, a mixed number, and an exact decimal (with repeating digits marked, e.g. 1/3 = 0.(3)). Arithmetic uses big integers, so denominators and results stay exact however large they grow. Everything runs locally in your browser.
