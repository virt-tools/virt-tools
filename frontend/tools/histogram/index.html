Frequency Distribution & Histogram

Bin a list of numbers into a frequency distribution with equal-width intervals and visualize it as an ASCII histogram. Enter a series, choose the number of bins, and the tool reports each bin's range, count, cumulative count, and relative frequency, plus an ASCII bar chart scaled to the tallest bin. Useful for exploring the shape of a dataset before reaching for a chart library. Runs locally in your browser.

Frequency table & histogram

The data is binned into k equal-width intervals spanning [min, max]: bin width w = (max − min) / k. Bin i holds values in [min + i·w, min + (i+1)·w), and the last bin is closed on the right (it includes max). Each bin reports its count, cumulative count, relative frequency (count / n), and an ASCII bar scaled so the tallest bin fills the available width. When all values are identical the width is zero and everything lands in the first bin. Values are parsed as numbers separated by commas, spaces, or newlines. Pairs with the Statistics Calculator and Moving Average tools. Everything runs locally — nothing leaves your browser.
