Vigenère Cipher

Encode and decode text with the classic Vigenère polyalphabetic cipher. A repeating key shifts each letter; case, numbers and punctuation pass through untouched. Runs entirely in your browser.

Each plaintext letter is shifted by the corresponding key letter (A=0 … Z=25), with the key repeating. Non-letters in the text are kept but skipped when advancing the key.
