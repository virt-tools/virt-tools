Image Filter & Convolution Kernel

Apply adjustable filters — brightness, contrast, saturation, hue rotate, grayscale, sepia, invert, and blur — and run a custom 3×3 convolution kernel (sharpen, edge detect, emboss, box blur, Gaussian, Sobel, Laplacian, or your own matrix). Adjust everything with live sliders and download the result as PNG. The image is processed on a local canvas — nothing is uploaded. Pairs with Image Duotone, Image Dithering, and Image Histogram.

3×3 Convolution kernel



CSS-style filters run first (brightness, contrast, etc.), then the 3×3 convolution kernel is applied. Kernel divisor scales the result; offset adds a bias (useful for emboss). Runs locally on a canvas; nothing is uploaded.
