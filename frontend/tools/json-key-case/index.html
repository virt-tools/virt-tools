JSON Key Case Converter

Recursively convert all object keys in a JSON document between camelCase, snake_case, kebab-case, PascalCase, CONST_CASE (SCREAMING_SNAKE), lowercase, or dot.case. Arrays and primitive values are preserved; only keys are renamed. Useful for normalizing API responses or config files. Everything runs locally in your browser.

Paste JSON and choose a target case.
