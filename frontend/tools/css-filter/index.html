CSS Filter Generator

Build a CSS filter declaration visually. Drag sliders for blur, brightness, contrast, grayscale, hue-rotate, invert, opacity, saturate, and sepia; the preview updates live and the ready-to-copy filter: string lists only the functions you've changed from their defaults. Reset clears everything. Runs entirely in your browser.

Preview

filter: none;

CSS filter applies graphical effects to an element. Values: blur(<px>), brightness(<number>) (1 = unchanged), contrast(<number>), grayscale(<0–1>), hue-rotate(<deg>), invert(<0–1>), opacity(<0–1>), saturate(<number>), sepia(<0–1>). The output string omits functions left at their defaults so it stays clean. Runs locally.
