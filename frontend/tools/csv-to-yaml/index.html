CSV to YAML Converter

Convert CSV data into YAML — each row becomes a list item (mapping) with the header row as keys. Handles quoted fields, embedded commas and newlines, and ragged rows. Choose the delimiter (auto, comma, semicolon, tab, pipe) and toggle the header row. Pairs with the YAML Converter and CSV to JSON. Everything runs locally in your browser.

Each CSV row becomes a list item. With headers, each item is a mapping (key: value). Without headers, keys are col1, col2, etc. Numeric values are left unquoted; strings needing YAML quoting are handled. Pairs with the YAML Converter and CSV to JSON. Runs locally.
