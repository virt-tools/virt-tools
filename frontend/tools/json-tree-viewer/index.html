JSON Tree Viewer

Paste JSON and navigate it as a collapsible tree. Click any node to expand or collapse it; click a key or value to see its JSON Pointer path and copy it to the clipboard. Each node shows its type, size, and a short preview. Search highlights matching keys and values, with prev/next navigation. Distinct from the JSON Formatter (which pretty-prints text) — this is an interactive structural explorer. Everything runs locally in your browser.
