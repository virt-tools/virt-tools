CSS Loader Generator

Pick a spinner style, tune its size, speed and colors, then copy the ready-to-paste HTML and CSS into your project. The preview runs the exact generated CSS, so what you see is what you get. Everything runs locally in your browser; nothing is uploaded.

Each style is pure CSS (keyframe animations, no JavaScript, no images). The class name is .loader and is scoped under .vt-loader so it won't clash with your existing styles — drop both the HTML and CSS in, or rename to taste. Colors accept any CSS value (hex, rgb, hsl, named). No network requests are made.
