Image to Emoji Art

Turn an image into emoji art — each pixel block maps to the emoji whose average color is closest. Pick a palette set (basic colors, flags, nature, food, faces, or full), set the output width, and copy or download the result. The image is processed on a local canvas — nothing is uploaded. Pairs with the ASCII Art, Braille Art, and Block Art generators.
