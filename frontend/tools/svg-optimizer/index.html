SVG Optimizer

Paste or upload an SVG and strip the cruft — comments, editor metadata (Inkscape/sodipodi/RDF), the XML prolog, empty groups — while minifying path coordinates and collapsing whitespace. The visible image is unchanged; the file is smaller. Everything runs locally in your browser; nothing is uploaded.

Click to upload an .svg file (or drop one here)

The optimizer is intentionally conservative: it removes only non-rendering cruft and rounds numeric coordinates to your chosen precision (trailing zeros stripped). Path d data is re-emitted with minimal separators (a separator is kept only where a number would otherwise run together). Text content inside <text>/<tspan> and the bodies of <style>/<script> are preserved verbatim, so the visible image is not affected. No network requests are made.
