Beaufort Cipher

Encode and decode text with the Beaufort cipher, a reciprocal polyalphabetic cipher devised by Sir Francis Beaufort (of the Beaufort wind scale). Like the Vigenère cipher it uses a repeating keyword, but with the key and plaintext reversed: each ciphertext letter is C = (K − P) mod 26, so the same operation encrypts and decrypts — applying the cipher twice with the same key returns the original text. Case, digits, and punctuation pass through untouched; the key advances only on letters. Runs entirely in your browser.

Each ciphertext letter is (key − plaintext) mod 26, with the key repeating. The cipher is reciprocal, so Encode and Decode are the same operation.

The math. The Beaufort cipher maps each plaintext letter P against a key letter K as C = (K − P) mod 26. Decryption is identical: P = (K − C) mod 26 = (K − (K − P)) mod 26 = P, so the cipher is reciprocal (self-inverse) — the same function with the same key both enciphers and deciphers. This differs from the Vigenère cipher, which uses C = (P + K) mod 26 for encryption and P = (C − K) mod 26 for decryption; the two ciphers coincide only in the special case of a single-letter key (where both reduce to a Caesar shift, but in opposite directions). Pairs with the Caesar, Vigenère, and Classical Ciphers tools. Everything runs locally — nothing leaves your browser.
