Beaufort Cipher

Encrypt and decrypt with the Beaufort cipher, a polyalphabetic substitution cipher that is a variant of the Vigenère cipher. While Vigenère adds the key to the plaintext, Beaufort subtracts the plaintext from the key. A fascinating property of the Beaufort cipher is that it is self-reciprocal: the same la-algorithme is used for both encryption and decryption. Everything runs locally in your browser.

The math. Let $P$ be the plaintext letter and $K$ be the key letter (both 0-25). The Beaufort cipher calculates the ciphertext letter $C$ as: $C = (K - P) \pmod{26}$. Because the formula is symmetric, decrypting $C$ using the same key $K$ yields: $P = (K - C) \pmod{26}$. For example, if $P=H(7)$ and $K=B(1)$, then $C = (1 - 7) \pmod{26} = -6 \pmod{26} = 20(U)$. Pairs with the Caesar, Vigenère, and Hill cipher tools. Everything runs locally — nothing leaves your browser.
