Braille Art Generator

Turn an image into Unicode braille art — each 2×4 pixel block becomes a single braille character (U+2800…U+28FF), producing a high-resolution text rendering. Pick a threshold (or dither), set the output width, and flip light/dark for the background. Copy or download the result. The image is read with FileReader and drawn to a local canvas — nothing is uploaded, and no network requests are made. Pairs with the ASCII Art generator.
