Unicode Block & Character Explorer

Browse the Unicode blocks — Basic Latin, Latin-1 Supplement, Greek and Coptic, Cyrillic, Arrows, Mathematical Operators, Box Drawing, Geometric Shapes, Dingbats, Emoji ranges, and more. Click a block to see its glyphs in a grid; click a glyph to inspect its code point, name, UTF-8 bytes, HTML entity, and copy it. Pairs with the Character & Unicode Inspector and the ASCII Table. Static reference data, runs locally in your browser.

Pick a block to browse its characters.
