CSV to Markdown Table

Convert CSV data into a GitHub-flavoured Markdown table. Handles quoted fields, embedded commas and newlines, ragged rows, and per-column alignment (left, center, right). Choose the delimiter (auto, comma, semicolon, tab, pipe) and whether the first row is a header. Pairs with the Markdown Table Generator and CSV Viewer. Everything runs locally in your browser.

Cells containing pipes are escaped with \|. Ragged rows are padded to the widest row. The separator row uses --- (left), :---: (center), or ---: (right) per the alignment choice. Pairs with the Markdown Table Generator and CSV Viewer. Runs locally.
