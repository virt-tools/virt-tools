Object fit

object-fit controls how an element's content (image, video) is resized to fit its container. fill stretches, contain fits entirely, cover fills and crops, none shows at natural size, scale-down picks the smaller of none or contain. object-position shifts the focal point. Pairs with CSS Aspect Ratio and Image Resizer. Runs locally.
