EXIF / Image Metadata Viewer

Open a JPEG or TIFF and read the EXIF metadata embedded in it — camera make & model, orientation, date taken, exposure (shutter speed, aperture, ISO, focal length), pixel dimensions and GPS coordinates. The file is read locally with FileReader; nothing is uploaded, and the parser makes no network requests.
