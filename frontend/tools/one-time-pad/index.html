One-Time Pad

Encrypt and decrypt with the One-Time Pad (OTP), the only mathematically unbreakable cipher when used correctly. It uses a truly random key (the "pad") that is at least as long as the message itself. Each letter of the message is combined with the corresponding letter of the pad (typically via modulo 26 addition), and the pad must be used only once and then destroyed. Everything runs locally in your browser.

The math. For a plaintext letter $P$ and key letter $K$, the ciphertext letter $C$ is $C = (P + K) \pmod{26}$. To decrypt, the process is reversed: $P = (C - K) \pmod{26}$. For the cipher to be unbreakable, the key must be: (1) truly random, (2) at least as long as the plaintext, (3) kept secret, and (4) never reused. If the key is shorter than the message, this becomes a Vigenère cipher, which is vulnerable to analysis. Pairs with the Caesar, Vigenère, and Hill cipher tools. Everything runs locally — nothing leaves your browser.
