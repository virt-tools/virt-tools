Paced Breathing Tool

Slow your breathing with a visual guide. The circle expands as you inhale, holds, contracts as you exhale — follow it at a steady pace. Pick a preset (Box, 4-7-8, Relax, Calm) or set your own phase lengths, then breathe along. Optional audio cues mark each phase change. Everything runs locally in your browser.

Ready
Press Start to begin

The guide cycles through inhale → hold in → exhale → hold out at the durations you set (a zero-length phase is skipped). The circle's size is interpolated smoothly across the cycle, so it grows, holds, and shrinks in time with each phase. Common patterns: Box (4-4-4-4) for focus, 4-7-8 for falling asleep, Relax (4-2-6) to unwind. Optional audio uses a short Web Audio beep. Runs locally — no data leaves your browser.
