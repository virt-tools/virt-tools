Image Watermark

Add a text or logo watermark to an image — place it in a corner, centered, or tiled across the whole picture, with adjustable opacity, rotation, size and color. Everything renders locally on a canvas; nothing is uploaded.

Click to choose an image
or drop one here, or paste (Ctrl/⌘+V). PNG, JPG, WebP, GIF, BMP.

Load an image to begin.

The watermark is drawn on an offscreen canvas at the image's full resolution, then scaled into the preview — so the download matches what you see. For tiled watermarks, spacing is a percentage of image width; rotation rotates each tile around its own center. No network requests are made.
