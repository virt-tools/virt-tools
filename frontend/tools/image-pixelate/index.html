Image Pixelator

Pixelate an image to create chunky, retro-style pixels. Set the pixel block size and optionally reduce the color palette for an 8-bit look. Choose from preset palettes (Game Boy, PICO-8, NES, CGA) or keep the original colors — just blocky. Adjust the pixel grid, then download as PNG. The image is processed on a local canvas — nothing is uploaded. Pairs with the Image Dithering and ASCII Art tools.

Click to choose, or drag & drop an image here



The block size controls how chunky each pixel is — larger values make bigger blocks. The palette option quantizes each block's average color to a limited set for a retro look. Output scale upscales the result so small pixel art exports at a usable size. Pairs with Image Dithering and ASCII Art. Runs locally.
