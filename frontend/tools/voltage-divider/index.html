Enter the values to compute the output voltage.

A voltage divider produces an output voltage (Vout) that is a fraction of its input voltage (Vin). The formula is: Vout = Vin × (R2 / (R1 + R2)). The total current is I = Vin / (R1 + R2). Note that the output voltage is only valid if the load connected to Vout has a much higher resistance than R2; otherwise, the load will "pull down" the output voltage.
