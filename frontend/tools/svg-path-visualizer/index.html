SVG Path Visualizer

Paste an SVG path d string and see it rendered with a breakdown of every command — M, L, H, V, C, S, Q, T, A, Z — plus the bounding box, total length, and command count. Toggle the control-point handles and the start/end markers. Great for learning, debugging, and reverse-engineering path data. Pairs with the SVG Editor and SVG Optimizer. Everything runs locally in your browser.

The path is parsed and rendered locally; nothing is uploaded. Bounding box and length use the browser's native SVGGeometryElement APIs. Control-handle display works for cubic (C/S) and quadratic (Q/T) Bézier segments. Pairs with the SVG Editor and SVG Optimizer.
