Image to ANSI Art

Turn an image into ANSI terminal art using 24-bit true color escape codes. Each pixel block maps to a \x1b[38;2;R;G;Bm color code with a half-block character (▀ or ▄), so two pixel rows fit in one text line — double the vertical resolution of basic block art. Copy the ANSI sequence to paste into a terminal, or download the .ans file. The image is processed on a local canvas — nothing is uploaded. Pairs with the Block Art and Braille Art generators.
