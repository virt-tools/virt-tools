GPA Calculator

Compute your grade-point average on the standard 4.0 scale. Enter each course as a letter grade (A, A−, B+, …, F) or as a percentage, set the credit hours, and the weighted GPA updates live. Supports plus/minus grades, credit weighting, and a percentage→letter conversion using a common 10-point scale. Runs entirely in your browser.

Add courses to compute your GPA.

The 4.0 scale: A+/A = 4.0, A− = 3.7, B+ = 3.3, B = 3.0, B− = 2.7, C+ = 2.3, C = 2.0, C− = 1.7, D+ = 1.3, D = 1.0, D− = 0.7, F = 0.0. GPA = Σ(points × credits) ÷ Σ(credits). Percentage mode maps 93+→A, 90→A−, 87→B+, 83→B, 80→B−, 77→C+, 73→C, 70→C−, 67→D+, 63→D, 60→D−, below 60→F. Courses with zero or blank credits are excluded from the GPA but still listed. This is a generic calculator — confirm your school's exact scale if it differs. Runs locally.
