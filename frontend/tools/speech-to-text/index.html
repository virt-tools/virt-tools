Speech to Text

Transcribe speech to text in your browser using the Web Speech API. Click the microphone, start speaking, and your words appear as you talk — no upload, no recording, no API key. Choose a language, toggle continuous recognition and interim results. Copy or download the transcript. Pairs with the Text to Speech tool. Everything runs locally in your browser; audio is processed by the browser's built-in speech engine and never sent to a server by this tool.

Click the microphone to start.

The Web Speech API is supported in Chrome, Edge, and Safari, and requires a network connection for the browser's speech engine (the audio is not sent by this tool — the browser handles it natively). Firefox support is limited. Interim results are shown in grey and replaced when finalized. Pairs with the Text to Speech tool. Runs locally in your browser.
