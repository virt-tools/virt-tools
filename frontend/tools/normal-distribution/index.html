Normal Distribution Calculator

Work with the normal (Gaussian) distribution. Enter a mean μ and standard deviation σ, then either find the cumulative probability P(X ≤ x) for a given value (with its z-score and density), invert a percentile to find the value x for a given probability, or compute the probability P(a ≤ X ≤ b) of an interval. The CDF uses the error function and the percentile inversion uses a high-accuracy rational approximation; results match standard normal tables to about six significant figures. Everything runs locally in your browser.
