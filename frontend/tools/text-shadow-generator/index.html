CSS Text Shadow Generator

Build layered CSS text-shadows with a live preview and copy-ready CSS. Stack multiple shadows per layer, or start from a preset — neon glow, retro 3D, soft, outline, emboss, or a stepped long shadow — and tweak from there. Everything runs in your browser; nothing is uploaded.

Shadow

Each layer is offset-x offset-y blur-radius color. text-shadow has no spread and no inset (those are box-shadow only). Layers render bottom-to-top, so the first listed sits on top. No network requests are made.
