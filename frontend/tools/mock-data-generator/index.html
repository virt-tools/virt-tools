Mock Data Generator

Generate realistic fake records — names, emails, addresses, companies, dates, and more — with a custom schema and an optional seed for reproducible output. Export as JSON, NDJSON, CSV, or SQL. Everything runs locally; no data leaves your browser.

Built-in data pools are small and English-ish for variety. Uses a seeded PRNG (mulberry32) when you set a seed, so the same seed + schema reproduces the same data — handy for fixtures. No network requests are made.
